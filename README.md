# Lawnlayer
Java game application built with gradle (USYD 2022_s1 coursework)


## Version
- Gradle 7.4.2
- Java 1.8.0


## How to use
- Edit level files (.txt) 
- Configs within config.json
```
gradle build
gradle run
gradle javadoc
```
